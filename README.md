# Library Management System (GEHU LMS)

## Overview
A web-based Library Management System designed to manage book records, users, and issue/return operations efficiently.  
The system improves record management and reduces manual workload through automation.

---

## Objective
- Digitize library operations  
- Improve record accuracy and accessibility  
- Reduce manual administrative effort  
- Provide quick search and reporting features  

---

## Key Features
- User authentication  
- Book issue and return management  
- Student and book record maintenance  
- Database-driven operations  
- Simple and user-friendly interface  

---

## Tools & Technologies
- HTML, CSS, Bootstrap  
- PHP  
- MySQL  

---

## Outcomes & Learnings
- Hands-on experience in full-stack development  
- Understanding of database-driven systems  
- Learned process automation and system design
